Project Name: Cool Sort

Project Descrition: The objective of this project is to demonstrate the working of a sorting algorithm 'CoolSort()' which is derived from insertion sort.

Installation:
Compile the project using command: 'make'
Run the project by using command: './coolSort'

Usage:
The project can be used to sort an array using cool sort. It allows of swapping of elements that are far apart thereby reducing time taken to perform sorting.
